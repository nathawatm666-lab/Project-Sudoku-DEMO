# 🧩 Java Sudoku Game

โปรแกรมเกมซูโดกุ (Sudoku) ที่พัฒนาด้วยภาษา Java (Swing) มาพร้อมกับส่วนติดต่อผู้ใช้ (GUI) ในโทนสี Dark Mode ที่ดูทันสมัยและสบายตา ตัวเกมใช้ระบบอัลกอริทึม **Backtracking** ในการสร้างกระดานแบบสุ่มและระบบเฉลยคำตอบอัตโนมัติ ทำให้ผู้เล่นสามารถเพลิดเพลินกับโจทย์ที่ไม่ซ้ำกันในทุกๆ ครั้งที่เริ่มเกมใหม่

## ✨ ฟีเจอร์เด่น (Features)

* **🎮 Modern UI & Animations:** หน้าต่างเมนูหลักมาพร้อมเอฟเฟกต์ตัวอักษรแบบคลื่น (Wave Animation) และโทนสีเกมที่ออกแบบมาเพื่อความสบายตา
* **🧠 Backtracking Algorithm:** ใช้ความรู้เรื่องโครงสร้างข้อมูลและอัลกอริทึม (Data Structures & Algorithms) ในการสุ่มสร้างโจทย์ (Generation) และหาคำตอบที่ถูกต้อง (Solver)
* **🎚️ ระดับความยาก 4 ระดับ:** สามารถเลือกเล่นได้ตั้งแต่ Easy, Medium, Hard ไปจนถึง Expert (ระบบจะสุ่มลบตัวเลขตามระดับความยาก)
* **⏱️ ระบบจับเวลา & นับข้อผิดพลาด:** มีนาฬิกาจับเวลาเพื่อท้าทายความสามารถ และมีระบบ Game Over หากกรอกตัวเลขผิดเกิน 5 ครั้ง
* **💡 ตัวช่วยเล่น (Helpers):**
    * **Hint:** สุ่มเฉลยตัวเลขที่ถูกต้องให้ 1 ช่อง
    * **Erase:** ลบตัวเลขที่กรอกผิด
    * **Solve:** ยอมแพ้และให้ระบบแสดงเฉลยทั้งกระดานทันที
* **⌨️ รองรับการใช้คีย์บอร์ด:** สามารถใช้ปุ่มลูกศร (Arrow Keys) เพื่อเลื่อนช่อง และปุ่ม Backspace/Delete เพื่อลบตัวเลขได้

## 🛠️ เครื่องมือและเทคโนโลยีที่ใช้ (Tech Stack)

* **Language:** Java
* **GUI Library:** Java Swing & AWT (ใช้ `CardLayout` สำหรับสลับหน้าจอ)
* **Algorithm:** Recursion & Backtracking

## 🚀 วิธีการติดตั้งและรันโปรแกรม (How to Run)

1.  ตรวจสอบให้แน่ใจว่าเครื่องของคุณติดตั้ง **Java Development Kit (JDK)** แล้ว
2.  ดาวน์โหลดหรือโคลน (Clone) โปรเจกต์นี้ลงในเครื่องของคุณ
3.  เปิด Terminal หรือ Command Prompt แล้วไปยังโฟลเดอร์ที่เก็บไฟล์ `SudokuGame.java`
4.  คอมไพล์โค้ดด้วยคำสั่ง:
    ```bash
    javac SudokuGame.java
    ```
5.  รันโปรแกรมด้วยคำสั่ง:
    ```bash
    java SudokuGame
    ```

## 📂 โครงสร้างการทำงานของโค้ด (Code Structure)

* `createMenuPanel()`: จัดการหน้าจอเมนูหลักและ Animation
* `createBoardPanel()`: สร้างตารางซูโดกุขนาด 9x9 และช่องย่อย 3x3 
* `generateSolution()` & `fillBoard()`: ใช้ Backtracking Algorithm สร้างตารางคำตอบที่สมบูรณ์
* `createPuzzle()`: สุ่มลบตัวเลขออกจากกระดานตามระดับความยากที่เลือก
* `checkWin()`: ตรวจสอบความถูกต้องเมื่อผู้เล่นกรอกตัวเลขครบทุกช่อง พร้อมแสดงเอฟเฟกต์ฉลองชัยชนะ
